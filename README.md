# Traffic-light-Controller

## Overview
A digital traffic light controller designed using sequential logic circuits and implemented in Proteus simulation software.

## Working Principle
The system cycles through traffic light states (Red → Yellow → Green) using clock pulses and timing control.

## Design Methodology
- State diagram design
- Truth table creation
- Sequential logic implementation
- Proteus simulation

## Components Used
- Logic gates
- Flip-flops (if used)
- Clock generator
- LEDs (Red, Yellow, Green)
- Resistors

## Simulation
The circuit was designed and tested in Proteus to verify correct timing and state transitions.
